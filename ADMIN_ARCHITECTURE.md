## Key Metrics Calculation

### USDC Deposited
```sql
SELECT SUM(amount) 
FROM onchain_deposits 
WHERE asset = 'USDC' 
  AND status IN ('confirmed', 'swept')
```

### USDC Offramped
```sql
SELECT SUM(quotes.crypto_amount)
FROM payouts
JOIN quotes ON payouts.quote_id = quotes.id
WHERE quotes.asset = 'USDC'
  AND payouts.status = 'success'
```

### NGN Paid Out
```sql
SELECT SUM(fiat_amount)
FROM payouts
WHERE status = 'success'
  AND currency = 'NGN'
```

### Platform Revenue
```sql
SELECT SUM(quotes.total_fee)
FROM payouts
JOIN quotes ON payouts.quote_id = quotes.id
WHERE payouts.status = 'success'
```
